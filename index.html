Phishing Email Detector

Hybrid rule-based + ML classification engine for phishing detection

Active Flagship
+

Case Study

Built to address one of the most common attack vectors in enterprise environments, this tool combines rule-based heuristics (65%) and machine learning (35%) in a hybrid detection pipeline across 8 distinct modules.

The classifier uses Naive Bayes via scikit-learn trained on labeled phishing datasets, while the rule engine evaluates header anomalies, sender spoofing patterns, suspicious link structures, and urgency language — mimicking how a SOC analyst would triage a suspicious email manually.

IMAP/SMTP integration allows the tool to connect directly to a live inbox and run classification on incoming mail in near real time.

Network Traffic Analyzer

Packet capture and anomaly detection for SOC triage workflows

In Progress
+

Case Study

Designed to replicate the packet-level visibility a SOC analyst needs during triage, this tool captures live network traffic and surfaces behavioral anomalies against an established baseline.

The analyzer performs protocol inspection across TCP, UDP, ICMP, and HTTP, flags unusual port activity, detects potential C2 beaconing patterns, and exports structured summaries for further investigation — bridging the gap between raw Wireshark captures and actionable alerts.

Malware Analysis Sandbox

Controlled environment for behavioral profiling of suspicious binaries

In Progress
+

Case Study

A controlled, isolated environment for detonating and analyzing suspicious executables without risk to the host system. Designed around the workflow of a malware analyst performing dynamic behavioral analysis.

Captures system call sequences, file system write events, registry key modifications, and outbound network callbacks in real time. Results are compiled into a standardized behavioral profile that maps to MITRE ATT&CK techniques where applicable, making findings immediately actionable for threat intel teams.
